# Heart Animation ❤️

A Python turtle graphics animation that creates and displays a heart shape using mathematical functions.

## Features
- Animated heart drawing
- Smooth curve using math functions
- Simple and clean visual output

## Technologies Used
- Python
- Turtle Graphics
- Math module

## How to Run
1. Install Python
2. Run:
   python heart_animation.py
